# xRaySocks5MTProto


# На EN сервере (основной):
```bash
chmod +x install_en_server.sh
```
```bash
./install_en_server.sh
```
**→ выбрать 1 (полная установка)**
**→ скопировать IP из /root/proxy_settings.html**

===========================================================================

# На RU сервере (relay): 

```bash
chmod +x install_ru_relay.sh
```
```bash
./install_ru_relay.sh
```
** → выбрать 1 (полная настройка)**
** → ввести IP EN сервера и порты**

===========================================================================

В клиентах подключайтесь к IP RU сервера, используя параметры из EN HTML-файла.

===========================================================================

# Текущее поведение по пунктам меню

| Пункт меню | EN скрипт | RU скрипт
|------------ |------------ |------------ 
| 1) Полная установка | ❌ Перегенерирует ВСЁ (новые UUID, ключи, пароли) | ✅ Предлагает использовать старые данные
| 2) Перезапуск | ✅ Использует сохранённые настройки | ✅ Использует сохранённые настройки
| 3) Перегенерация/Изменить | ✅ Генерирует новые ключи | ✅ Запрашивает новые данные
| 4-5) Статус/Настройки | ✅ Читает из сохранённого конфига | ✅ Читает из сохранённого конфига
| 6) Удаление | ❌ Удаляет всё (включая конфиг) | ❌ Удаляет всё (включая конфиг)
